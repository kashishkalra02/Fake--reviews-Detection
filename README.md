# 🔍 Fake Review Detection

An AI-powered Fake Review Detection system that identifies fraudulent and misleading online reviews using Machine Learning and Natural Language Processing (NLP). The system analyzes review text and classifies it as either **Genuine** or **Fake**, helping users make informed decisions and improving trust in online platforms.

## 🚀 Features

* Detects fake and genuine reviews
* NLP-based text preprocessing
* Sentiment analysis
* Machine Learning classification
* Real-time review prediction
* User-friendly interface
* Fast and accurate review verification

## 🛠️ Technologies Used

* Python
* Machine Learning
* Natural Language Processing (NLP)
* Scikit-learn
* Pandas
* NumPy
* Matplotlib
* Flask / Streamlit

## 📖 How It Works

1. User enters a review.
2. The review text is preprocessed.
3. NLP techniques extract meaningful features.
4. The trained machine learning model analyzes the review.
5. The system predicts whether the review is **Fake** or **Genuine**.

## ⚙️ Installation

1. Clone the repository:

```bash
git clone https://github.com/your-username/Fake-Review-Detection.git
```

2. Navigate to the project directory:

```bash
cd Fake-Review-Detection
```

3. Install dependencies:

```bash
pip install -r requirements.txt
```

4. Run the application:

```bash
python app.py
```
